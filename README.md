Overview
This repository contains 8 beginner-friendly Python mini-projects aimed at practicing programming fundamentals, GUI development, file handling, API usage, and basic AI logic. Each project is designed for learning, demonstration, and exam/viva preparation.
Projects List
1. Currency Converter App
Objective: Convert amounts between different currencies.
Features:
CLI or Tkinter GUI
Mocked/static exchange rates or API
Input: amount, source currency, target currency
Output: converted amount
2. Expense Tracker (CLI)
Objective: Track daily/monthly expenses.
Features:
Add expenses by category & date
Store in CSV/JSON
Monthly totals per category
Bar/Pie chart using matplotlib
3. Library Book Management System (CLI)
Objective: Manage library books and student transactions.
Features:
Add/remove books
Issue & return handling
Fine calculation for overdue books
Store data in JSON/dictionary
4. Quiz App (Tkinter GUI)
Objective: Interactive MCQ quiz for learning and practice.
Features:
Multiple choice questions
Next/Previous navigation
Score calculation
Optional JSON-based question storage
5. Weather App (Tkinter GUI + API)
Objective: Fetch real-time weather data for a city.
Features:
Input city name
Display temperature, humidity, condition
Uses OpenWeatherMap API
Optional weather icons
6.To-Do List App (CLI)
Objective: Manage tasks and prioritize work.
Features:
Add, edit, delete tasks
Mark tasks as completed
Save/load tasks using JSON
Optional priority & due date
7. Password Generator Tool (CLI)
Objective: Generate secure passwords to improve security.
Features:
user-defined length, digits, special characters
Randomized secure passwords
Optional save to file/clipboard
8. Simple Chatbot (CLI, Rule-Based)
Objective: Basic AI assistant for answering predefined queries.
Features:
Input user messages
Keyword-based responses using if-else/dictionaries
Optional: NLTK for text preprocessing

Can simulate website assistant
